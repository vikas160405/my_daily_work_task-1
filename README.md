# Rule-Based Chatbot

This project is a simple web-based Rule-Based Chatbot built using Python and Flask. The chatbot responds to user inputs using predefined rules and basic keyword matching.

Overview

The chatbot is designed to demonstrate how rule-based conversational systems work. It processes user input, applies conditional logic, and returns appropriate responses based on matched keywords.

This project does not use machine learning. Instead, it focuses on understanding conversation flow and backend-frontend integration.

Features

- Web-based user interface
- Predefined responses using if-else logic
- Case-insensitive keyword matching
- Basic conversation handling
- Clean and simple user interface

Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

How to Run

1. Install dependencies:
   pip install flask

2. Run the application:
   python app.py

3. Open your browser and go to:
   http://127.0.0.1:5000

Purpose

The purpose of this project is to understand the fundamentals of rule-based chatbots, request handling in Flask, and integration between backend and frontend components.
